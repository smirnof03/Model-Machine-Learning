# Model Machine Learning

## Deskripsi
Repository ini berisi implementasi model machine learning yaitu KLASIFIKASI JENIS TANAMAN BERDASARKAN PARAMETER LINGKUNGAN DENGAN LOGISTIC REGRESSION
untuk memenuhi tugas mata kuliah Machine Learning.

## Metode
Model dibuat menggunakan algoritma machine learning
dengan tahapan preprocessing, training, dan evaluasi.

## Tools

- Jupyter Notebook
- Scikit-learn
